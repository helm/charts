Memcached can be accessed via port 11211 on the following DNS name from within your cluster:
{{ template "memcached.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local

If you'd like to test your instance, forward the port locally:

  POD_NAME=$(kubectl get pods --namespace {{ .Release.Namespace }} -l "app={{ template "memcached.fullname" . }}" -o jsonpath="{.items[0].metadata.name}")
  kubectl port-forward $POD_NAME {{ .Values.memcached.port }}

In another tab, attempt to set a key:

  echo -e 'set mykey 0 60 5\r\nhello\r' | nc localhost {{ .Values.memcached.port }}

You should see:

  STORED
