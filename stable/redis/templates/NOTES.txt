Redis can be accessed via port 6379 on the following DNS name from within your cluster:
{{ template "fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local

{{ if .Values.redisPassword }}
To get your password run:

    REDIS_PASSWORD=$(printf $(printf '\%o' `kubectl get secret --namespace {{ .Release.Namespace }} {{ template "fullname" . }} -o jsonpath="{.data.redis-password[*]}"`))
{{ end }}

To connect to your Redis server:

1. Run a Redis pod that you can use as a client:

   kubectl run {{ template "fullname" . }}-client --namespace {{ .Release.Namespace }} --rm --tty -i {{ if .Values.redisPassword }}--env REDIS_PASSWORD=$REDIS_PASSWORD {{ end }}--image {{ .Values.image }} -- bash

2. Connect using the Redis CLI:

  redis-cli -h {{ template "fullname" . }}{{ if .Values.redisPassword }} -a $REDIS_PASSWORD{{ end }}
