bitcoind RPC can be accessed via port {{ .Values.RPCPort }} on the following DNS name from within your cluster:
{{ template "bitcoind.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local

To connect to bitcoind RPC:

1. Run an Ubuntu pod that you can use as a client:

    kubectl run -i --tty ubuntu --image=ubuntu:16.04 --restart=Never -- bash -il

2. Install the CURL:

    $ apt-get update -y && apt-get install -y curl

3. Connect using the bitcoind cli, then provide your password:
    $ curl --user <rpc_user>:<rpc_password> -k http{{- if .Values.ssl.enabled }}s{{- end }}://{{ template "bitcoind.fullname" . }}:{{ .Values.RPCPort }} --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getinfo", "params": [] }' -H 'content-type: text/plain;'
